IHC Icons Replacement Instructions
=====================================

The following icon files need to be replaced with IHC-branded versions:

1. 32x32.png - 32x32 pixel IHC logo
2. 128x128.png - 128x128 pixel IHC logo
3. 128x128@2x.png - 256x256 pixel IHC logo (high-DPI)
4. icon.ico - Multi-resolution ICO file with IHC logo (16x16, 24x24, 32x32, 64x64)

Colors to use:
- Primary: IHC Blue (hsl(217, 78%, 27%))
- Secondary: IHC Light Blue (hsl(217, 91%, 85%))
- Background: White or transparent

These should match the design style of the SVG logos created in /src/assets/
